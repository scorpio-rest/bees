# Galaga Next - 經典小蜜蜂遊戲

這是一個使用 React + TypeScript + Vite 打造的現代化經典街機「小蜜蜂」(Galaga) 復刻版。

## 🚀 特色
- **精美視覺**：使用 HTML5 Canvas 渲染，包含動態星空背景與霓虹特效。
- **流暢體驗**：優化的遊戲循環與碰撞檢測系統。
- **響應式設計**：支援多種顯示尺寸。
- **準備部署**：內建 Netlify 設定，可快速部署到雲端。

## 🕹️ 操作說明
- **方向鍵 (← / →)**：控制戰機左右移動。
- **空白鍵 (Space)**：發射子彈。
- **目標**：消滅不斷逼近的外星蜜蜂陣列。每過一關，敵人的速度與射擊頻率都會提升。

## 🛠️ 開發與建構

### 本地開發
```bash
npm install
npm run dev
```

### 專案建構
```bash
npm run build
```

## 🌐 部署到 Netlify
本專案已包含 `netlify.toml`。您可以將此儲存庫連結至 Netlify，它會自動完成建構與部署。
